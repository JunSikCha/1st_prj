# 🔑 핵심기능

### 1. 차량 정비 예약

> 사용자로 로그인 시 차량을 등록할 수 있고, 등록한 차량의 정비를 예약할 수 있습니다. <br/>
> 과거에 정비를 받은 내역이나 예약한 내역을 확인할 수 있습니다.

### 2. 예약 관리, 재고 관리

> 관리자로 로그인 시 사용자들의 예약 내역을 관리할 수 있습니다. <br/>
> 그 외에 정비소 내 부품 등의 재고를 관리할 수 있고, 발주 처리 등의 작업을 처리할 수 있습니다.
